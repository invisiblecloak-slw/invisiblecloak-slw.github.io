0078-subsets

Xiu Di / 2021-05-01 Leetcode

问题描述

  • 给定整数数组nums，数组中的元素互不相同。
  • 返回所有可能的子集（幂集）。
  • 解集不能包含重复的子集。可以按任意顺序返回解集。

示例1

输入: nums = [1,2,3]

输出: [[], [1], [2], [1,2], [3], [1,3], [2,3], [1,2,3]]

示例2

输入: nums = [0]

输出: [[], [0]]

提示

  • 1 <= nums.length <= 10
  • -10 <= nums[i] <= 10
  • nums中的所有元素互不相同

代码

class Solution
    def subsets(self, nums: List[int]) -> List[List[int]]:
        res = list()
        def backtrack(index: int, tmp: List[int]) -> None:
            res.append(tmp[:])
            for i in range(index, len(nums)):
                tmp += [nums[i]]
                backtrack(i+1, tmp)
                tmp.pop()
        backtrack(0, [])
        return res
