🚩 0142-linked list cycle ii

Xiu Di / 2021-05-03 Leetcode

问题描述

给定一个链表，返回链表开始入环的第一个节点。如果链表无环，则返回null。

为了表示给定链表中的环，使用整数pos来表示链表尾连接到链表中的位置（索引从0开始）。 如果pos是-1，则在该链表中没有环。

示例1:

输入: head = [3,2,0,-4], pos = 1

输出: 返回索引为1的链表节点

解释: 链表中有一个环，其尾部连接到第二个节点。

示例2:

输入: head = [1,2], pos = 0

输出: 返回索引为0的链表节点

解释: 链表中有一个环，其尾部连接到第一个节点。

示例3:

输入: head = [1], pos = -1

输出: 返回null

解释: 链表中没有环。

提示;

  • 链表中节点数目范围在[0, 104]内。
  • -105 <= None.val <= 105。
  • pos的值为-1或链表中的一个有效索引。
  • 不允许修改给定的链表。
  • 考虑使用O(1)空间复杂度解决此问题。

代码

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, x):
#         self.val = x
#         self.next = None

class Solution:
    def move(self) -> bool:
        if not self.rabbit.next or not self.rabbit.next.next:
            return False

        self.rabbit = self.rabbit.next.next
        self.tortoise = self.tortoise.next
        return True

    def detectCycle(self, head: ListNode) -> ListNode:
        if not head:
            return None
        
        self.tortoise = self.rabbit = head
        
        # Floyd判环法
        if not self.move():
            return None
        while self.rabbit != self.tortoise:
            if not self.move():
                return None

        # 寻找环口节点
        self.tortoise = head
        while self.tortoise != self.rabbit:
            self.tortoise = self.tortoise.next
            self.rabbit = self.rabbit.next

        return self.tortoise

分析

快慢指针

  • 设快慢指针分别为rabbit和tortoise。二者每时间间隔分别移动两步、一步。
  • 若链表无环，按照上述设定，rabbit显然更早成为null指针。
  • 若链表有环，有下述结论：
    • rabbit指针与tortoise指针必相遇。
    • 当tortoise指针与rabbit指针相遇时，tortoise`指针尚未在环中周游一圈。
  • 上述结论证明如下：
    • 考虑相对运动问题。当tortoise指针运动到环入口时，设此时rabbit指针与tortoise指针的顺时针距离为c，rabbit指针距离环入口的距离为b，表头距离环入口的距离为a。
    • 由于tortoise指针与rabbit指针的速度分别为1步、2步，可视为tortoise指针静止，而rabbit指针以1步每时间间隔的速度运动。因此经过c时间间隔后，tortoise指针与rabbit指针相遇。
    • 由c in [0, n-1]，其中n为环的长度。则rabbit指针与tortoise指针相遇时，tortoise指针尚未在环中周游一圈。
  • 相遇时设rabbit指针已经绕环p圈。
    • 当p = 0时有：环的长度为a。
    • 当p >= 1时有：
      • tortoise指针运动的距离为a + b。
      • rabbit指针运动的距离为2(a + b) = a + p( b + c ) + b。
      • 因此有a = (p - 1)(b + c) + c。
      • 此时置tortoise指针和rabbit指针的速度均为1步每时间间隔，则两指针将在环入口相遇。
