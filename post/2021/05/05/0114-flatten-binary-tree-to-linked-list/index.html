🚩 0114-flatten binary tree to linked list

Xiu Di / 2021-05-05 Leetcode

问题描述

给定二叉树的根节点root，将它展开为一个单链表：

  • 展开后的单链表应该同样使用TreeNode，其中right子指针指向链表中下一个结点，而左子指针始终为null。
  • 展开后的单链表应该与二叉树先序遍历顺序相同。

示例1:

       1                  1
      / \                  \
     2   5        =>        2
    / \   \                  \
   3   4   6                  3
                               \
                                4
                                 \
                                  5
                                   \
                                    6

输入: root = [1,2,5,3,4,null,6]

输出: [1,null,2,null,3,null,4,null,5,null,6]

示例2:

输入: root = []

输出: []

示例3:

输入: root = [0]

输出: [0]

提示:

  • 树中结点数在范围[0, 2000]内
  • -100 <= Node.val <= 100

代码

递归

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val=0, left=None, right=None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def flatten(self, root: TreeNode) -> None:
        def flattenHelper(root: TreeNode) -> TreeNode:
            if not root:
                return None
            if not root.left:
                flattenHelper(root.right)
                return root
            elif not root.right:
                root.right = flattenHelper(root.left)
                root.left = None
                return root
            else:
                flattenHelper(root.left)
                tail = root.left
                while(tail.right):
                    tail = tail.right
                tail.right = flattenHelper(root.right)
                root.left, root.right = None, root.left
                return root

        flattenHelper(root)

类Morris算法

  • 将左子树插入到右子树的位置
  • 将原来的右子树接到左子树的最右侧节点
  • 继而考虑新的右子树的根节点，一直重复上边的过程，直到新的右子树为null
class Solution:
    def flatten(self, root: TreeNode) -> None:
        while root:
            # 如果左子树为空，则直接考虑下一个结点
            if not root.left:
                root = root.right
            else:
                # 找到左子树的最右侧结点
                tail = root.left
                while tail.right:
                    tail = tail.right

                # 将右子树插入到左子树的最右侧结点
                tail.right = root.right
                
                # 将左子树置为右子树，并将左子树置空
                root.left, root.right = None, root.left

                # 考虑下一个节点
                root = root.right
