Xiu Di / 2021-05-14 Leetcode

问题描述

给定整数n，求恰由n个节点组成且节点值从1到n互不相同的二叉搜索树的个数。

示例1:

输入: n = 3

输出: 5

示例2:

输入: n = 1

输出: 1

提示: 1 <= n <= 19

代码

class Solution:
    def numTrees(self, n: int) -> int:
        dp = [0] * (n + 1)
        dp[0], dp[1] = 1, 1
        for amount in range(2, n+1):
            for root in range(1, amount + 1):
                left_amount = root - 1
                right_amount = amount - root
                dp[amount] += dp[left_amount] * dp[right_amount]
        return dp[n]
