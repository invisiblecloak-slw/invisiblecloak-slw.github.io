0560-subarray sum equals k

Xiu Di / 2021-05-20 Leetcode

问题描述

给定整数数组和整数k，找到该数组中和为k的连续子数组的个数。

示例1:

输入: nums = [1,1,1], k = 2

输出: 2, [1,1] 与 [1,1] 属于两种不同的情况。

说明:

  • 数组长度为[1, 2e4]
  • 数组中元素范围为[-1e3, 1e3], 整数k范围为[-1e7, 1e7]

代码

前缀和

class Solution:
    def subarraySum(self, nums: List[int], k: int) -> int:
        hashTable = collections.defaultdict(int)

        hashTable[0] = 1

        prefixSum, cnt = 0, 0
        for num in nums:
            prefixSum += num
            if hashTable[prefixSum - k]:
                cnt += hashTable[prefixSum - k]
            hashTable[prefixSum] += 1
        return cnt
