0581-shortest unsorted continuous subarray

Xiu Di / 2021-05-22 Leetcode

问题描述

给定整数数组nums，找出其中的一个连续子数组S，若对S进行升序排列，那么数组nums将会变为升序排列。

找出符合题意的最短子数组，并输出其长度。

示例1:

输入: nums = [2,6,4,8,10,9,15]

输出: 5

解释: 只需要对[6,4,8,10,9]进行升序排列，那么整个表就会变成升序排列。

示例2:

输入: nums = [1,2,3,4]

输出: 0

示例3:

输入: nums = [1]

输出: 0

提示:

  • 1 <= nums.length <= 1e4
  • -1e5 <= nums[i] <= 1e5
  • 考虑时间复杂度为O(n)的解决方案

代码

class Solution:
    def findUnsortedSubarray(self, nums: List[int]) -> int:
        sz = len(nums)
        if sz == 1: return 0
        # 从右向左遍历数组，若存在后续元素值小于当前元素值，那么当前元素应纳入排序范畴
        minElement = nums[-1]
        left = -1
        for idx in range(sz-1, -1, -1):
            if minElement < nums[idx]:
                left = idx
            else:
                minElement = nums[idx]
        # 从左向右遍历数组，若存在之前元素值大于当前元素值，那么当前元素应纳入排序范畴
        right = -1
        maxElement = nums[0]
        for idx in range(sz):
            if maxElement > nums[idx]:
                right = idx
            else:
                maxElement = nums[idx]

        return 0 if left == -1 else right - left + 1
