0338-counting bits

Xiu Di / 2021-05-17 Leetcode

问题描述

给定非负整数num，对于0<=i<=num范围中的每个数字i，计算其二进制数中的1的数目并将它们作为数组返回。

示例1:

输入: 2

输出: [0, 1, 1]

示例2:

输入: 5

输出: [0, 1, 1, 2, 1, 2]

进阶:

  • 考虑在线性时间复杂度**O(n)**内完成。
  • 要求空间复杂度为O(n)。

代码

class Solution:
    def countBits(self, num: int) -> List[int]:
        count = [0] * (num + 1)
        for i in range(1, num + 1):
            if i & 1:
                count[i] = count[i-1] + 1
            else:
                count[i] = count[i >> 1]
        return count
