0394-decode string

Xiu Di / 2021-05-23 Leetcode

问题描述

给定一个经过编码的字符串，返回它解码后的字符串。

编码规则为:k[encoded_string]，表示其中方括号内部的encoded_string正好重复k次。其中k为正整数。

输入字符串总是有效的。输入字符串中没有额外的空格，且输入的方括号总是符合格式要求。

原始数据中的数字只表示重复的次数k，不会出现像3a或2[4]的输入。

示例1:

输入: s = "3[a]2[bc]"

输出: aaabcbc

示例2:

输入: s = "3[a2[c]]"

输出: "accaccacc"

示例3:

输入: s = "2[abc]3[cd]ef"

输出: "abcabccdcdcdef"

示例4:

输入: s = "abc3[cd]xyz"

输出: "abccdcdcdxyz"

代码

class Solution:
    def decodeString(self, s: str) -> str:
        numStack = list()
        strStack = list()
        
        curStr, multiplier = '', 0
        for c in s:
            if '0' <= c <= '9':
                multiplier = 10 * multiplier + ord(c) - ord('0')
            elif 'a' <= c <= 'z' or 'A' <= c <= 'Z':
                curStr += c
            elif c == '[':
                numStack.append(multiplier)
                strStack.append(curStr)
                curStr, multiplier = '', 0
            elif c == ']':
                curStr *= numstack.pop()
                strStack[-1] += curStr
                curStr = strStack.pop()

    return curStr
