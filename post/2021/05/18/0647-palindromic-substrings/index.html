0647-palindromic substrings

Xiu Di / 2021-05-18 Leetcode

问题描述

给定一个字符串，计算该字符串中有多少个回文子串。

具有不同开始位置或结束位置的子串，即使由相同字符组成，也会被视作不同的子串。

示例1:

输入: "abc"

输出: 3

示例2:

输入: "aaa"

输出: 6

提示:

  • 输入字符串长度不超过1000。

代码

class Solution:
    def countSubstrings(self, s: str) -> int:
        n = len(s)
        isParlindrome = [[False] * n for _ in range(n)]
        cnt = 0
        for j in range(n):
            for i in range(j+1):
                # base case

                    isParlindrome[i][j] = True

                    cnt += 1
        return cnt
