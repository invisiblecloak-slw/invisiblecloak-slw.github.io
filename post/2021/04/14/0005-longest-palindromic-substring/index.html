0005-longest palindromic substring

Xiu Di / 2021-04-14

最长回文子串

给定字符串s，找到s中的最长回文子串。

示例1：

输入: s = “babad”

输出: “bab"或"aba”

示例2：

输入: s = “cbbd”

输出: “bb”

示例3：

输入: s = “a”

输出: “a”

示例4：

输入: s = “ac”

输出: “a”

提示:

  • 1 <= s.length <= 1000
  • s仅由数字和英文字母（大写和/或小写）组成

代码

动态规划法

class Solution:
    def longestPalindrome(self, s: str) -> str:
        n = len(s)
        # 记录dp过程中发现的最长回文子串
        start, stride = 0, 0
        # step = 0, 默认为True
        dp = [[True for j in range(n)] for i in range(n)]
        # step = 1, 另一种构建块，进行判定
        for in in range(n - 1):
            if s[i] == s[i+1]:
                start, stride = i, 1
            else:
                dp[i][i+1] = False
        # step in [2, n-1], 进行判定
        for step in range(2, n):
            for i in range(n - step):
                if not dp[i + 1][i + step - 1] or s[i] != s[i + step]:
                    dp[i][i + step] = False
                else:
                    start, stride = i, step
        return s[start: start + stride + 1]

中心扩散法

class Solution:
    def longestPalindrome(self, s: str) -> str:
        n, maxlen = len(s), 1
        start, end = 0, 0
        # 以单个元素为构建块
        for i in range(n):
        l, r = i - 1, i + 1
        while l >= 0 and r < n:
            if s[l] != s[r]: break
            elif maxlen < r - l + 1:
                maxlen, start, end = r - l + 1, l, r
            l, r = l - 1, r + 1
        # 以两个元素为构建块
        for i in range(n - 1):
            if s[i] != s[i+1]: continue
            else:
                if maxlen < 2: maxlen, start, end = 2, i, i + 1
                l, r = i - 1, i + 2
                while l >= 0 and r < n:
                    if s[l] != s[r]: break
                    elif maxlen < r - l + 1:
                        maxlen, start, end = r - l + 1, l, r
                    l, r = l - 1, r + 1
        return s[start: end + 1]

关于中心扩散法的重构(LC题解)

class Solution:
    def expandAroungCenter(self, s, left, right):
        while left >= 0 and right < len(s) and s[left] == s[right]:
            left -= 1
            right += 1
        return left + 1, right -1
    
    def longestPalindrome(self, s: str) -> str:
        start, end = 0, 0
        for i in range(len(s)):
            left1, right1 = self.expandAroundCenter(s, i, i)
            left2, right2 = self.expandAroundCenter(s, i, i+1)
            if right1 - left1 > end - start:
                start, end = left1, right1
            if right2 - left2 > end - start:
                start, end = left2, right2
        return s[start: end + 1]

分析

  • Python借助列表构建二维数组的语句形如:
    • array = [[0 for j in range(nj)] for i in range(ni)]
    • j和i分别指定数组的列数和行数。
  • 嵌套循环中，break跳出最内层循环。
