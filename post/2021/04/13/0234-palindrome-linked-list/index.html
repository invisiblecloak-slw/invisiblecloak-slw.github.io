0234-palindrome linked list

Xiu Di / 2021-04-13 Leetcode

问题描述

请判断一个链表是否为回文链表

示例1:

输入: 1->2->2->1

输出: true

示例2:

输入: 1->2

输出: false

提示:考虑使用O(n)时间复杂度和O(1)空间复杂度。

代码

将链表中的元素复制到数组中，借助数组元素的随机访问特性，使用双指针法判定是否为回文链表

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val = 0, next = None):
#         self.val = val
#         self.next = next
class Solution(self, head : ListNode) -> bool:
    ptr, array = head, list()
    while ptr:
        array.append(ptr.val)
        ptr = ptr.next
    lptr, rptr = 0, len(array) - 1
    while lptr < rptr:
        if array[lptr] != array[rptr]: return False
        lptr, rptr = lptr + 1, rptr - 1
    return True

参考逆序打印链表，设置前向指针，与逆向递归栈的节点比较元素值

class Solution:
    def isPalindrome(self, head: ListNode) -> bool:
        self.front = head
        def helper(head: ListNode):
            if not head: return True
            if helper(head.next) and head.val == self.front.val:
                self.front = self.front.next
                return True
            return False
        return helper(head)

显式使用栈

class Solution:
    def isPalindrome(self, head: ListNode) -> bool:
        if not head: return True
        stack, ptr = list(), head
        while ptr:
            stack.append(ptr)
            ptr = ptr.next
        while head:
            if head.val != stack.pop().val: return False
            head = head.next
        return True

结合翻转链表算法

class Solution:
    def reverseList(self, head: ListNode) -> bool:
        if not head or not head.next: return head
        rhead = self.reverseList(head.next)
        head.next.next, head.next = head, None
        return rhead
    
    def ifPalindrome(self, head: ListNode) -> bool:
        if not head or not head.next: return True
        slow = fast = head
        while fast.next and fast.next.next:
            slow, fast = slow.next, fast.next.next
        # 如果链表长度是奇数，那么slow指针位于链表正中央
        # 如果链表长度是偶数，那么slow指针位于链表前半部分最后节点
        rhead = self.reverseList(slow)
        ptr, rptr = head, rhead
        while rptr != slow and ptr != rptr:
            if ptr.val != rptr.val: return False
            ptr, rptr = ptr.next, rptr.next
        return True

分析

对于结合翻转链表的方法，须注意：

  • rhead = self.reverseList(slow) 反转后半部分链表，其尾节点应为slow而非slow.next
  • 如果采用迭代方式实现链表翻转函数，那么本问题可以在O(1)的空间复杂度内解决。
