0017-letter combinations of a phone number

Xiu Di / 2021-04-17 Leetcode

题目描述

给定一个仅包含数字2-9的字符串，返回所有它能表示的字母组合。

给定数字到字母的映射如下（与电话按键相同）。

示例1

输入: digits = “23”

输出: [“ad”, “ae”, “af”, “bd”, “be”, “bf”, “cd”, “ce”, “cf”]

示例2

输入: digits = ""

输出: []

示例3

输入: digits = “2”

输出: [“a”, “b”, “c”]

提示:

  • 0 <= digits.length <= 4
  • digits[i]是范围['2', '9']的一个数字
  • 答案可以按任意顺序返回

代码

BFS

class Solution:
    def letterCombinations(self, digits: str) -> List[str]:
        digit = [str(x) for x in range(2, 10)]
        alphabet = ['abc', 'def', 'ghi', 'jkl', 'mno', 'pqrs', 'tuv', 'wxyz']
        d = dict(zip(digit, alphabet))
        queue = list()
        cnt = 0
        for digit in digits:
            if not cnt:
                for alphabet in d[digit]:
                    queue.append(alphabet)
            else:
                for _ in range(cnt):
                    prefix = queue.pop(0)
                    for alphabet in d[digit]:
                        queue.append(prefix + alphabet)
            cnt = len(queue)
        return queue

DFS

class Solution:
    def DFS(self, digits: str, comb: str) -> List[str]:
        if not digits:
            if comb: self.res.append(comb)
        else:
            for alphabet in self.d[digits[0]]:
                self.DFS(digits[1:], comb + alphabet)
    def letterCombinations(self, digits: str) -> List[str]:
        digit = [str(x) for x in range(2, 10)]
        alphabet = ['abc', 'def', 'ghi', 'jkl', 'mno', 'pqrs', 'tuv', 'wxyz']
        self.d = dict(zip(digit, alphabet))
        self.res = list()
        self.DFS(digits, '')
        return self.res
