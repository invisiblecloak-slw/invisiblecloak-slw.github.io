VIM Shorcuts

Xiu Di / 2021-04-06 Shortcuts

跳转命令

shift + g: 跳转到文件结尾

gg: 跳转到文件开头

:10: 跳转到第10行

删除命令

:1,10d: VIM模式下删除1～10行的内容

选中命令

v+j/k/h/l: 进入VISUAL模式，文本选中。

对于选中的文本可执行以下操作：

d: 剪切

y: 复制

p: 粘贴

0: 选中当前行，光标位置到行首

$: 选中当前行，光标位置到行尾

撤销与恢复

u: 撤销上一次操作

:redo: 恢复

Level 1

命令 功能 命令 功能
h Move cursor left :wq :x ZZ Save changes and close file
l Move cursor right x Delete character at cursor
j Move cursor down i Insert at cursor
k Move cursor up I Insert at beginning of line
:q Close file a Append at cursor
:q! Close file, don't save changes A Append at end of line
:q Save changes to file ESC ctrl+[ Exit insert mode

Level 2

命令 功能 命令 功能
dw Delete word 3e Go to end of third word ahead
d$ D Delete to end of line d2w Delete two words
w Next word dd Delete entire line
$ Go to end of current line 2dd Delete two lines
^ Go to beginning of text on current line u Undo last change
0 Go to beginning of current line U Undo changes on entire line
2w Go two words forward CTRL+R Redo changes
