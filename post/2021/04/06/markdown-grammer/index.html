表格

markdown制作表格，使用|分隔不同单元格，-分隔表头和其他行。

语法格式为：

| 表头 | 表头 |

| ---- | ---- |

| 单元 | 单元 |

| 单元 | 单元 |

可以设置表格的对齐方式：

  • -:设置单元格内容和标题栏居右对齐
  • :-设置单元格内容和标题栏居左对齐
  • :-:设置单元格内容和标题栏居中对齐
