0198-house robber

Xiu Di / 2021-04-20 Leetcode

问题描述

你是一个专业的小偷，计划偷窃沿街的房屋。每间房内都藏有一定的现金，影响你偷窃的唯一制约因素就是相邻的房屋装有相互连通的防盗系统，如果两间相邻的房屋在同一晚上被小偷闯入，系统会自动报警。

给定一个代表每个房屋存放金额的非负整数数组，计算你不触动警报装置的情况下，一夜之内能够偷窃到的最高金额。

示例1:

输入: [1,2,3,1]

输出: 4

示例2:

输入: [2,7,9,3,1]

输出: 12

提示:

  • 0 <= nums.length <= 100
  • 0 <= nums[i] <= 400

代码

class Solution:
    def rob(self, nums: List[int]) -> int:
        if not nums: return 0
        if len(nums) == 1: return nums[0]
        if len(nums) == 2: return max(nums[0], nums[1])
        pre, cur = nums[0], max(nums[0], nums[1])

分析

本问题可以用动态规划的思想进行求解。

  • 考虑第k间房屋。
    • 若选择盗窃第k间房屋，则可偷窃到最大金额G1为第k间房屋存放金额与前k-2间房屋可偷窃到的最大金额之和。
    • 若选择不偷窃第k间房屋，则可偷窃得到的最大金额G2为前k-1间房屋可偷窃到的最大金额。
  • 因此前k间房屋可偷窃得到的最大金额为max(G1, G2)。
  • 小偷一夜之间能够偷窃到的最高金额可表示为前n间房屋可偷窃得到的最大金额。
