0022-generate parentheses

Xiu Di / 2021-04-18

问题描述

设计函数，生成所有可能的并且有效的括号组合。

数字n代表生成括号的对数。

示例1:

输入: n = 3

输出: ["((()))", “(()())”, “(())()”, “()(())”, “()()()"]

示例2:

输入: n = 1

输出: ["()"]

提示: 1 <= n <= 8

代码

DFS1

class Solution:
    def helper(self, comb = '', lcnt = 0, matched = 0) -> None:
        if matched == self.n:
            self.res.append(comb)
        elif matched == lcnt:
            self.helper(comb + '(', lcnt + 1, matched)
        elif lcnt < self.n:
            self.helper(comb + '(', lcnt + 1, matched)
            self.helper(comb + ')', lcnt, matched + 1)
        else:
            self.helper(comb + ')', lcnt, matched + 1)
    def generateParenthesis(self, n: int) -> List[str]:
        self.res = list()
        self.n = n
        self.helper()
        return self.res

DFS2

class Solution:
    def DFS(self, comb = '', cnt = 0, max_cnt = 0, score = 0, max_score = 0):
        if cnt == max_cnt:
            if score == 0: self.res.append(comb)
        else:
            if score + 1 <= max_score:
                self.DFS(comb + '(', cnt + 1, 2 * self.n, score + 1, self.n)
            if score - 1 >= 0:
                self.DFS(comb + ')', cnt + 1, 2 * self.n, score - 1, self.n)

    def generateParenthesis(self, n: int) -> List[str]:
        self.res = list()
        self.n = n
        self.DFS(max_cnt = 2 * self.n, max_score = self.n)
        return self.res

分析

  • 可以设定初值为0的分值。每添加一个(分值+1，添加)分值-1。
    • 一个合法的括号组合，最终得分为0。
    • 整个DFS过程中，得分范围为[0, n]。
      • 得分不超过n意味着不能添加数量超过n的左括号
      • 得分不为负数，意味着每个右括号必有一个左括号与之匹配。
