0206-reverse linked list

Xiu Di / 2021-04-11 Leetcode

问题描述

反转一个简单链表。

示例:

输入: 1->2->3->4->5->NULL

输出: 5->4->3->2->1->NULL

提示:

可以使用递归和迭代两种方式反转链表。

代码

递归

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        if not head or not head.next: return head
        rhead = self.reverseList(head.next)
        head.next.next, head.next = head, None
        return rhead

迭代

# Definition for singly-linked list.
# class ListNode:
#     def __init__(self, val=0, next=None):
#         self.val = val
#         self.next = next
class Solution:
def reverseList(self, head: ListNode) -> ListNode:
    if not head or not head.next: return head
    prev, cur, Next = None, head, head.next
    while cur:
        cur.next, prev, cur = prev, cur, Next
        if Next: Next = Next.next
    return prev
