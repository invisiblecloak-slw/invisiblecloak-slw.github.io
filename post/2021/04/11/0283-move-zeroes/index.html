0283-move zeroes

Xiu Di / 2021-04-11 Leetcode

问题描述

给定数组nums，编写一个函数将所有0移动到数组的末尾，同时保持非零元素的相对顺序。

示例

输入: [0,1,0,3,12]

输出: [1,3,12,0,0]

说明:

  • 必须在原数组上操作，不能拷贝额外的数组

  • 尽量减少操作次数

代码

双指针 - 1

class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        count = 0
        for num in nums:
            if num:
                nums[count] = num
                count += 1
        nums[count:] = [0] * len(nums[count:])

双指针 - 2

class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        i = 0
        for j in range(len(nums)):
            if nums[j]:
                nums[i], nums[j] = nums[j], nums[i]
                i += 1

注: 在每一次循环之前，i左边的元素均非零。

  • 初值i=0，显然成立。
  • 每次循环中，若nums[j] == 0，则nums[i]保持不变。若nums[j] != 0，交换后i增一，仍满足上述性质。
  • 且i左侧的非零元素的相对顺序维持不变。

借助filter

class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        n = len(nums)
        nums = list(filter(lambda x: x != 0, nums))
        nums.extend([0] * (n - len(nums)))

借助remove

# 效率较低
class Solution:
    def moveZeroes(self, nums: List[int]) -> None:
        if nums.count(0):
            for _ in range(len(nums)):
                nums.remove(0)
                nums.append(0)

分析

filter

Python的内建函数filter()用于过滤序列。

与map()类似，filter()接收一个函数和一个序列。filter()将传入的函数依次作用于序列的每个元素，并根据函数返回值为True还是False决定保留还是丢弃该元素。

例如，只保留list中的奇数可以写作: list(filter(lambda x: x % 2 == 1, [1, 2, 4, 5, 6, 9]))

  • filter()函数的关键在于正确实现筛选函数。
  • filter()函数返回的是一个Iterator，也就是一个惰性序列。若要强迫filter()完成计算结果，需要用list()函数获得所有结果并返回一个列表。

index Vs. count & remove Vs. pop

  • index(): 用于从列表中找出某个值第一个匹配项的索引位置，如果没有找到匹配项则抛出异常。用法为list.index(obj,[, start[, end]])，其中start和end是可选项。

  • count(): 用于统计某个元素在列表中的出现次数。若该元素在列表中不存在，则返回0。用法为list.count(obj)。

  • remove(): 用于移除列表中某个元素的首个匹配项，如果没有找到匹配项则抛出异常。用法为list.remove(obj)。该函数返回值为None。

  • pop(): 按照提供的索引值参数删除列表中的相应元素，并返回该元素。用法为list.pop(index)。

append Vs. extend Vs. +

  • append: append操作可以向列表中添加任意元素，如列表、字典、字符串等。

  • extend: extend操作可以扩展不同类型的对象，将对象内容加入原列表中。如:

    • In[1]: s = [1,2,3]
      • Out[1]: s = [1,2,3]
    • In[2]: s.extend([4])
      • Out[2]: s = [1,2,3,4]
    • In[3]: s.extend('abc')
      • Out[3]: s = [1,2,3,4,'a','b','c']
    • In[4]: s.extend({'age':12, 'height':180})
      • Out[4]: s = [1,2,3,4,'a','b','c','age','height']

  • +: +操作是对两个同类型变量的运算，不改变原有变量，返回两变量内容的拼接值。

for循环与enumerate

考虑代码段：

nums = [0, 1, 0, 3, 12]
for num in nums:
    if num == 0:
        nums.remove(num)
        nums.append(0)

在五次循环中，num的值分别为:0, 0, 12, 0, 0

考虑代码段：

nums = [0, 1, 0 ,3 ,12]
for i, num in enumerate(nums):
    if not num:
        nums.pop(i)
        nums.append(0)

在五次循环中，num的值分别为:0, 0, 12, 0, 0

可以在Python可视化网站中查看变量和内存的变化情况。

Python列表推导中使用if-else

Python列表推导共有两种形式：

  • x for x in data if condition
  • expression_1 if condition else expression_2 for x in data

应用场景将列表中的某些元素乘以某一倍数k可以用列表推导的方式完成。

  • In[1]: nums = [1, 3, 12]
    • Out[1]: nums = [1, 3, 12]
  • In[2]: nums = [2 * num for num in nums]
    • Out[2]: nums = [2, 6, 24]
  • In[3]: nums = [num if i < 1 else num * 2 for i, num in enumerate(nums)]
    • Out[3]: nums = [2, 12, 48]

注: nums *= 2将返回[1, 3, 12, 1, 3, 12]而非[2, 6, 24]。
