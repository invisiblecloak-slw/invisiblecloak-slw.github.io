0448-find all numbers disappeared in an array

Xiu Di / 2021-04-11

找到所有数组中消失的数字

给定一个范围在 1 <= a[i] <= n(n = 数组大小)的整型数组，数组中的元素一些出现了两次，另一些只出现一次。

找到所有在[1,n]范围之间没有出现在数组中的数字。

考虑不使用额外空间的时间复杂度为O(n)的算法。

示例

输入: [4,3,2,7,8,2,3,1]

输出: [5,6]

代码

O(1)空间复杂度

class Solution:
    def findDisappearedNumbers(self, nums: List[int]) -> List[int]:
        res = []
        n = len(nums)
        for num in nums: nums[(num-1) % n] += n
        for i, num in enumerate(nums):
            if num <= n: res.append(i+1)
        return res

class Solution:
    def findDisappearedNumbers(self, nums: List[int]) -> List[int]:
        res = []
        n = len(nums)
        for num in nums:
            if nums[abs(num)-1] > 0: nums[abs(num)-1] *= -1
        for i, num in enumerate(nums):
            if num > 0: res.append(i+1)
        return res

O(n)空间复杂度

class Solution(object):
    def findDisappearedNumbers(self, nums):
    counter = set(nums)
    N = len(nums)
    res = []
    for i in range(1, N + 1):
        if i not counter:
            res.append(i)
    return res

分析

  • 比较显然的一种方法是：对1-N的数字进行遍历，判断每个数字是否在数组中存在。
    • 可以使用set数据结构，其查找的时间复杂度为O(1)
  • 对于数组的原地操作，考虑到长度为N的数组可以用来统计1-N各数字出现的次数，因此可以原地修改数组实现计数。
    • 考虑引入额外信息，又能无损恢复原始数据的操作对，如取反和abs，加N和取模
