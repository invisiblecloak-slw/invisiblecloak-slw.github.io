0543-diameter of binary tree

Xiu Di / 2021-04-12

问题描述

给定一棵二叉树，计算它的直径长度。一棵二叉树的直径长度是指任意两个节点路径长度中的最大值。这条路径可能穿过也可能不穿过根节点。

示例

输入:

    1
   / \
  2   3
 / \
4   5

输出: 3

解释: 路径为[4,2,1,3]或[5,2,1,3]。

注意:两节点之间的路径长度是以它们之间边的数目表示。

代码

# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, val = 0, left = None, right = None):
#         self.val = val
#         self.left = left
#         self.right = right
class Solution:
    def calDepth(self, root: TreeNode) -> int:
        if not root: return 0
        else:
            left_depth, right_depth = self.calDepth(root.left), self.calDepth(root.right)
            self.diameter = max(self.diameter, left_depth + right_depth)
            return 1 + max(left_depth, right_depth)
    def diameterOfBinaryTree(self, root: TreeNode) -> int:
        self.diameter = 0
        self.calDepth(root)
        return self.diameter

分析

本问题与求解节点深度问题同构。在解决求解节点深度问题的同时，可以顺便解决本问题。
