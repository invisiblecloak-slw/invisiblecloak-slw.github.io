0461-hamming distance

Xiu Di / 2021-04-12

问题描述

两个整数之间的汉明距离指的是这两个数字对应二进制位不同的位置的数目。

给出两个整数x和y，计算它们之间的汉明距离。

示例

输入: x = 1, y = 4

输出: 2

解释:

1 (0 0 0 1)

4 (0 1 0 0)

注意: 0 <= x, y < 231

代码

Brian W. Kernighan & Dennis M. Ritchie

class Solution:
    def hammingDistance(self, x, y):
        xor = x ^ y
        cnt = 0
        while xor:
            cnt += 1
            xor &= xor - 1
        return cnt

Circular shift

class Solution:
    def hammingDistance(self, x, y):
        xor = x ^ y
        cnt = 0
        while xor:
            if xor & 1:
                cnt += 1
        return cnt

分析

本问题体现了位运算中关于计1的技巧。
