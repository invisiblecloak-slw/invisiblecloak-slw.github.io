0049-group anagrams

Xiu Di / 2021-04-27

问题描述

给定一个字符串数组，将字母异位词组合在一起。字母异位词指字母相同，但排序不同的字符串。

示例:

输入: [‘eat’, ‘tea’, ‘tan’, ‘ate’, ‘nat’, ‘bat’]

输出: [ [‘ate’,‘eat’,‘tea’], [‘nat’, ‘tan’], [‘bat’] ]

说明:

  • 所有输入均为小写字母。
  • 不考虑答案输出的顺序。

代码

以排序后的字符串作为键

class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        hashTable = collections.defaultdict(list)
        for word in strs:
            hashTable[''.join(sorted(word))].append(word)
        return list(hashTable.values()])

以字符串中各字母的频次统计作为键

class Solution:
    def groupAnagrams(self, strs: List[str]) -> List[List[str]]:
        hashTable = collections.default(list)
        for word in strs:
            cnt = [0] * 26
            for ch in word:
                cnt[ord(ch) - ord('a')] += 1
            hashTable[tuple(cnt)].append(word)
        return list(hashTable.values())

分析

  • collections.defaultdict(cls)在不存在的键被查找时，返回cls类型相应的默认值。
    • int返回0。list返回[]。str返回''。set返回set()。
  • 内置dict方法setdefault('key', 'defaultvalue')提供了defaultdict的等价行为。
    • 可以写成dict.setdefault('key', []).append(value)。
