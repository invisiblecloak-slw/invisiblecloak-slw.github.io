0070-climbing stairs

Xiu Di / 2021-04-08

问题描述

假设你正在爬楼梯。需要n阶你才能到达楼顶。

每次你可以爬1或2个台阶。你有多少种不同的方法可以爬到楼顶呢？

示例1:

输入: 2

输出: 2

解释: 有两种方法可以到达楼顶。

  • 1阶 + 1阶

  • 2阶

示例2:

输入: 3

输出: 3

解释:

  • 1阶 + 1阶 + 1阶

  • 1阶 + 2阶

  • 2阶 + 1阶

注意:

给定n是一个正整数。

代码

# 递归，超时
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2: 
            return n
        else:
            return self.climbStairs(n-1) + self.climbStairs(n-2)

# 递归，用字典规避递归过程中的重复计算
class Solution:
    d = dict(zip(range(3),range(3)))
    def climbStairs(self, n: int) -> int:
        if n in Solution.d: 
            return Solution.d[n]
        else:
            Solution.d[n] = self.climbStairs(n-1) + self.climbStairs(n-2)
            return Solution.d[n]

# 迭代
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2: return n
        prev, cur = 1, 2
        for i in range (3, n+1):
            prev, cur = cur, prev + cur
        return cur

分析

  • 解法3中使用类变量，为所有实例化对象所共享。调用方法为classname.var。

  • 递归与字典结合，可以避免部分重复计算。
