0009-palindrome number

Xiu Di / 2021-04-23

问题描述

给定整数x，如果x是一个回文整数，返回true，否则返回false。

示例1:

输入: x = 121

输出: true

示例2:

输入: x = -121

输出: false

示例3:

输入: x = 10

输出: false

提示: -231 <= x <= 231 - 1

代码

class Solution:
    def isPalindrome(self, x: int) -> bool:
        if x < 0: return False
        if x % 10 == 0 and x != 0: return False
        reverse_x = 0
        while x > reverse_x:
            reverse_x = 10 * reverse_x + x % 10
            x //= 10
        return x == reverse_x or x == reverse_x // 10
