0020-Valid Parentheses

Xiu Di / 2021-04-07

问题描述

给定一个只包括'(',')','{','}','[',']'的字符串s，判断字符串是否有效。

有效字符串需满足：

  • 左括号必须用相同类型的右括号闭合。
  • 左括号必须以正确的顺序闭合。

示例1:

输入: s = “()”

输出: true

示例2:

输入: s = “()[]{}”

输出: true

示例3:

输入: s = “(]”

输出: false

示例4:

输入: s = “([)]”

输出: false

示例5:

输入: s = “{[]}”

输出:true

提示:

  • 1 <= s.length <= 104

  • s仅由括号()[]{}组成

代码

class Solution:
    def isValid(self, s: str) -> bool:
        if len(s) % 2 == 1: return False
        dictionary = {'(':')', '{':'}', '[':']'}
        stack = []
        for ch in s:
            if ch in dictionary: 
                stack.append(ch)
            elif not stack or dictionary[stack.pop()] != ch:
                    return False 
        return not stack

分析

  • 栈可以用于括号匹配
  • 用字典存储相应的括号对
  • 列表可以用于实现栈
    • L[-1]访问栈顶元素
    • L.pop()删除并返回栈顶元素
  • not [] 将返回True，可用于判定列表是否为空
